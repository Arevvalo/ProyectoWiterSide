🌐 Proyecto Web

Este proyecto está construido utilizando tecnologías web fundamentales como HTML, CSS y JavaScript. Cada una de estas tecnologías juega un papel clave en la estructura, el diseño y la funcionalidad del sitio.

🌐 Tecnologías utilizadas:

📝HTML (HyperText Markup Language)

Se utiliza para definir la estructura básica de la página web. Con HTML, se crean los elementos fundamentales como encabezados, párrafos, imágenes, enlaces y formularios, que conforman el contenido visible y la jerarquía de la información.

🎨CSS (Cascading Style Sheets)

Es responsable de la apariencia visual del sitio. A través de CSS, se personalizan aspectos como los colores, las tipografías, los márgenes, el espaciado y la disposición de los elementos en la página. CSS también permite hacer que el diseño sea responsivo, adaptándose a diferentes tamaños de pantalla y dispositivos.

⚙️JavaScript

Aporta interactividad y dinamismo al sitio. Con JavaScript, se pueden implementar características como menús desplegables, animaciones, validación de formularios, y actualizaciones de contenido sin necesidad de recargar la página. JavaScript también se utiliza para la manipulación del DOM (Document Object Model), lo que permite modificar elementos de la página de forma dinámica en respuesta a la interacción del usuario.

Este enfoque basado en HTML, CSS y JavaScript permite desarrollar sitios web modernos, funcionales y visualmente atractivos, garantizando una experiencia de usuario óptima.
